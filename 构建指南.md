# QClaw ADB APK 快速构建指南

## 🎯 目标

将mi.py改造成Android APK,支持Android 9~16,使用Flet框架。

## 📦 方案一: GitHub Actions自动构建 (推荐)

### 步骤:

1. **创建GitHub仓库**
   ```powershell
   # 在桌面创建仓库
   cd C:\Users\24905\Desktop
   git init qclaw_adb_repo
   cd qclaw_adb_repo
   
   # 复制项目文件
   Copy-Item -Recurse C:\Users\24905\Desktop\qclaw_adb_apk\* .
   
   # 提交代码
   git add .
   git commit -m "Initial commit"
   git branch -M main
   
   # 连接GitHub仓库 (替换为你的仓库地址)
   git remote add origin https://github.com/YOUR_USERNAME/qclaw_adb.git
   git push -u origin main
   ```

2. **等待构建完成**
   - GitHub会自动开始构建
   - 大约10-15分钟后完成
   - 在Actions标签页查看进度

3. **下载APK**
   - 构建完成后,在Releases页面下载
   - 或在Actions页面下载artifacts

## 📦 方案二: WSL2本地构建

### 步骤:

1. **启用WSL2**
   ```powershell
   # 管理员PowerShell执行
   wsl --install -d Ubuntu-22.04
   ```

2. **进入WSL2**
   ```bash
   # 安装依赖
   sudo apt update
   sudo apt install -y git zip unzip openjdk-17-jdk python3-pip autoconf libtool pkg-config libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev automake
   
   # 安装Flet
   pip3 install flet
   
   # 进入项目目录
   cd /mnt/c/Users/24905/Desktop/qclaw_adb_apk
   
   # 构建APK
   flet build apk
   ```

3. **复制APK到桌面**
   ```bash
   cp build/*.apk /mnt/c/Users/24905/Desktop/
   ```
